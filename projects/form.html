Questions & Answers

1. The form tag has an "action" attribute. What does it do?

Answer: The action attribute tells the form **where the form data will be sent** when the user submits the form (a URL or server location).

2. The form tag has a "method" attribute. What does it do?

Answer: The method attribute determines **how** the form data is sent to the server — usually GET or POST.

3. What is the difference between a "POST" and a "GET"?

Answer: GET sends data through the URL and is mainly used for retrieving data, while POST sends data securely inside the HTTP request body and is used for submitting sensitive or long data.
