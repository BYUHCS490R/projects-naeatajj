Internet

A global system of interconnected computer networks that use standardized communication protocols to exchange information, giving people accessto share and create knowledge in real time.

Web

The Web is an information system on the Internet that uses websites and webpages that are connected by hyperlinks that allows people to access and share files with each other.

How are the Internet and Web different?

The Internet is the global system that carries signals everywhere whereas the Web which is the mind's organized thoughts where those signals are structured into meaningful ideas, stories and knowledge. Without the Internet, the Web couldn't exist. But without the Web, the Internet would still exist.

ASCII and Unicode

ASCII is a character encoding system that represents English letters, numbers, punctuation and some control symbols using numbers. Unicode is a universal character encoding standard that assigns a unique number to every character in virtually every writing, system, symbol set and emoji.

What is UTF-8

A UTF-8 is a way of storing and transmitting Unicode characters using sequences of 1 to 4 bytes. It is the most widely used character encoding on the Internet.

URL

A URL is like a home address for knowledge on the Web. The URL guides your web browser to the exact location of information.

Domain Name

Domain Name is like a nickname for knowledge on the Web. It is easy for people to remember instead of using long strings of numbers.

HTTP

The HTTP is the set of rules that computers use to talk to each other when sharing web pages over the Internet.

HTTPS

HTTPS is a secure version of HTTP. It's the set of rules that computers use to safely exchange web pages and data over the Internet.

IP Address

An IP Address is a unique numerical label assigned to every device connected to the Internet. It tells other computers where to find that device.

DNS

DNS is the system that translates human friendly domain names into IP addresses so that computers can understand.

W3C Recommendations

W3C Recommendations are like the rulebooks for building and sharing knowledge on the Web. They make sure websites and web applications work consistently for everyone and everywhere.

XML, XHTML, HTML, HTML5

HTML: The basic language of the Web used to create web pages.

HTML5: The latest version of HTML.

XHTML: A stricter version of HTML that follows XML rules.

XML: A language used to store and transport data.
