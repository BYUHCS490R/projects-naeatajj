JavaScript Questions

1. What is JavaScript? How did it get its name?

JavaScript is a high-level programming language used to make web pages interactive. It was originally called Mocha, then LiveScript. Marketing teams later renamed it "JavaScript" to ride the popularity of Java at the time, even though the languages are not closely related.

2. What are the two ways you can run JavaScript?

(1) Inside the web browser (client-side JavaScript).
(2) On a server using Node.js (server-side JavaScript).

3. What was the most popular front-end JS library?

The most popular front-end JavaScript library in the 2000s and 2010s was jQuery.

4. What is the most popular front-end JS library today?

Today the most popular front-end library is React, created by Facebook. It is widely used for building user interfaces and single-page applications.
