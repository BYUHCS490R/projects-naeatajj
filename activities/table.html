Engagement - Tables

Questions & Answers

1. When do you use a table?

You use a table when you need to organize data into rows and columns. Tables are best used when the information needs to be compared in a structured grid format, such as schedules, pricing charts, or statistical data.

2. When do you not use a table?

Do not use a table for page layout or general positioning. Tables should not be used to create the structure of a webpage because modern CSS tools (Flexbox and Grid) are designed for layout and work better across devices.

3. When do you need to do extra work for table accessibility?

Extra accessibility steps are needed when the table contains large or complex data. This includes adding a caption, proper header cells using <th>, and using scope="col" or scope="row" so that screen readers can interpret the table correctly.
