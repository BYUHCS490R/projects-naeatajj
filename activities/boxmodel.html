Box Model Questions

  1. What is the difference between block tags and inline tags?

    2. Block tags create a block-level element that starts on a new line and takes up the full width available. Inline tags do not start on a new line; they only take up as much width as needed.

  2. What is the margin, border, and padding?

    3. Margin: The space outside the border it separates an element from other elements. Border: The line or edge that goes around the padding and content. Padding: The space between the content and the border it pushes the content inward.

  3. What are units of measurement you can use in CSS?

    4. You can use absolute or relative units

  4. When would you use a figure tag?

    5. Use the figure tag when you have media content like an image, diagram, code snippet, or chart that goes with a caption or explanation.
