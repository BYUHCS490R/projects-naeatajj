Questions & Answers

1. What are different ways to style your form?

You can style forms using several CSS techniques such as: padding, margin, border-radius, box-shadow, fonts, colors, background-color, grid, and flexbox layouts. You can even use hover and focus states to highlight input fields when the user interacts with them.

2. How does flex help for both Desktop and Mobile phones?

Flexbox helps forms resize automatically. On desktops, the inputs can sit side-by-side (row layout), and on mobile screens, the direction can switch to column so elements stack vertically. This makes your webpage responsive without extra code.
