Questions & Answers

1. What is AJAX?

AJAX stands for Asynchronous JavaScript and XML. It allows a webpage to send or receive data from a server without refreshing the entire page. This makes websites smoother, faster, and more interactive.

2. Why do we put JavaScript in a separate file?

We put JavaScript in a separate file because:

  • It keeps the HTML cleaner and easier to read
  • JavaScript can be reused on multiple pages
  • Browsers cache JS files, making sites load faster
  • It follows best practice (separate HTML, CSS, JS)
