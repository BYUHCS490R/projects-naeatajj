Questions

1. Explain the difference between inline, embedded and external CSS?

  1. Inline CSS is written directly inside an HTML element's style attribute.
  2. Embedded CSS is written inside a style tag within the HTML.
  3. External CSS is stored in a separate .css file and linked with in the .

2. Explain why you should only use external CSS.

  1. Promotes consistency (all pages use the same stylesheet).
  2. Makes websites easier to maintain (update one file instead of editing many).
  3. Improves performance (browsers can cache the CSS file).
  4. Keeps HTML clean and organized (separation of content and design).

3. What is a selector used for?

A selector tells CSS which HTML elements to style.
