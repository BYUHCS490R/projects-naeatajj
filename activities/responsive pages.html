Class Questions & Answers

1. Why do we need a viewport meta tag?

The viewport meta tag ensures that the page displays correctly on all devices by controlling layout scaling and width. Without it, pages often look zoomed-out or distorted on mobile.

2. When you select the same tag twice, and one has "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

The one that appears last in the CSS file takes priority because of the **cascade** in Cascading Style Sheets (CSS). The most recent rule overrides earlier ones if they target the same element.

3. How do you change the CSS for a web page displayed on a phone?

You use a **media query** in your CSS. For example: 

@media screen and (max-width: 600px) {
  body {
    font-size: 16px;
    flex-direction: column;
  }
}

This adjusts the page style only when the screen width is 600px or smaller (like on phones).
